What is CodeConcise?

CodeConcise is a Graph RAG (Retrieval-Augmented Generation) platform that:

  • 1. Ingests Legacy Code: Takes a complete codebase as input
  • 2. Builds Knowledge Graph: Creates a semantic graph of all code relationships, dependencies, and patterns
  • 3. Enriches with Metadata: Uses LLM to add business context, purpose, and relationships to each code element
  • 4. Exposes APIs: Provides intelligent query interfaces to extract information

Why This Matters

Traditional code analysis tools only understand syntax. CodeConcise understands:

  • ‚úì Business Purpose: What each component does for the business
  • ‚úì Relationships: How components interact and depend on each other
  • ‚úì Patterns: Recurring architectural and design patterns
  • ‚úì Capabilities: High-level business capabilities the system provides

4

TOOLS: Execute Queries & Write Documents ‚≠ê

What happens: LLM executes tools based on its decision from MODEL phase

üîÑ Information Gathering Loop (Iterative):

‚Ä¢ Query 1: "What is the business purpose and stakeholders?" (minimal params)
‚Ä¢ Query 2: Based on response 1, ask follow-up: "What are the key workflows?" (escalate if needed)
‚Ä¢ Query 3: Based on responses 1-2, ask: "What are the technical details?" (escalate if needed)
‚Ä¢ Query 4: Based on responses 1-3, ask: "What are risks and improvements?" (escalate if needed)

5

MODEL ‚Üí TOOLS Loop Repeats Until Ready

What happens: After each CodeConcise query, control returns to MODEL phase

‚ùì Need more information? ‚Üí Loop back to TOOLS for more queries
‚úÖ Have enough context? ‚Üí Proceed to document generation
